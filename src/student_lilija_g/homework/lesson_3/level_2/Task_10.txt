Строка представляет собой последовательность символов.
Для работы со строками в Java определен класс String, который предоставляет ряд методов для манипуляции строками.
Физически объект String представляет собой ссылку на область в памяти, в которой размещены символы.

- Метод length() — возвращает длину строки в Java.

Сравнить две строки:
- с помощью метода compareTo()
- с помощью метода compareToIgnoreCase()
- с помощью метода equals()
- с помощью оператора ==


Для оптимальной работы приложения JVM делит память на область (stack) и область (heap).
Всякий раз, когда мы объявляем новые переменные, создаем объекты или вызываем новый метод, JVM выделяет память для этих операций в (stack) или в (heap).
...
Пул строк (string pool) в Java, это некий пул (или список) объектов класса String, который хранится в специальной области (Heap).
Это сделано для того, чтобы оптимизировать выделение памяти и хранение строк, ускорить и оптимизировать работу с ними.
Когда создаем строку с тем же значением, то сначала происходит поиск по пулу строк, и если там уже найдена такая строка,
- то этой строке присваивается ссылка на уже ранее созданный объект:

String strOne = "HELLO";
String strTwo = "HELLO";

strOne - попадает в пул строк. Когда создаем другую строку с таким же значением, то сначала происходит поиск по пулу строк.
И если там уже найдена такая строка - то strTwo присваивается ссылка на ранее созданный объект.

Команда String strTwo = "HELLO"; не создает другой объект строки, а происходит поиск по пулу строк.

Можно создать объект строки явно:

String strTwo = new String("HELLO"); - объект будет создан в области (heap). Поиск по пулу строк не будет происходить.
