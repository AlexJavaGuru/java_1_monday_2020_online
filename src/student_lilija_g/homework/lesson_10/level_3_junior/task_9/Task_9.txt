Найдите в интернете информацию про класс java.util.Optional.
Для чего применяются объекты этого класса?
Какие два состояния могут быть у объектов этого класса?
Как создаются экземпляры данного класса?
Как можно проверить в каком состоянии находится экземпляр данного класса?
Как можно получить объект, который находится внутри Optional?

Ответ.

Для чего применяются объекты этого класса:

Класс Optional призван помочь разработчикам в обработке NullPointerException.
Optional не дает никакой гарантии избавления от NullPointerException,
но с Optional это действия становятся быстрее и проще.
Optional помогает придать большую информативность коду, повысить его читабельность.
==================================================================================
Какие два состояния могут быть у объектов этого класса:
    - null
    _ != null

==================================================================================
Как создаются экземпляры данного класса:

Пустой Optional объект
    - Optional<Person> optionalPerson = Optional.empty();

Optional объект с ненулевым значением
    - Optional<Person> optionalNonNull = Optional.of(somePerson);

Optional объект с возможностью нулевого значения
    - Optional<Person> optionalNullable = Optional.ofNullable(somePerson);

==================================================================================
Как можно проверить в каком состоянии находится экземпляр данного класса:

Метод isPresent() класса java.util.Optional в Java используется,
чтобы узнать, присутствует ли значение в этом экземпляре Optional.
Если в этом необязательном экземпляре нет значения, этот метод возвращает false,
иначе true.

==================================================================================
Как можно получить объект, который находится внутри Optional:

    - Метод get(), если объект существует.
    - Если его нет, то возникнет ошибка NoSuchElementException.
